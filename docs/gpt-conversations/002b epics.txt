Stories:

🧠 What Actually Changed (aka what will hurt later)
1. You introduced time-based logic
Season
Wave
Priority phase

That means:

bugs won’t be obvious
testing gets harder
Copilot will absolutely guess wrong without help
2. You introduced state transitions

Reservation:

Active → Released → Reassigned

That’s not CRUD anymore. That’s a workflow.

3. You introduced cross-entity constraints

Reservation depends on:

Performance capacity
Wave phase
Subscription limit
Other users (released tickets)

That’s the part where systems go from “works locally” to “why is production on fire?”
